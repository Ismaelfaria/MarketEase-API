## 💻 Sobre o projeto


Projeto desenvolvido durante a **[Estudos sobre Rest API]** de [Desenvolvimento de Software].

O MarketEase-API é um projeto com o objetivo de simular um Sistema de Cadastro de Produtos correlacionando com a sua embalagem.

---

## ⚙️ Funcionalidades

- [x] Funcionalidade:
  - [x] Criar um Produto
  - [x] Atualizar um Produto
  - [x] Deletar um Produto
  - [x] Visualizar um/os Produtos

- [x] Funcionalidade 2:
  - [x] Adicionar a embalagem dos Produto
